Supported lists

HTML supports different types of lists—ordered, and unordered lists.

Ordered lists

An ordered list looks like that:
  1. London
  2. Paris
  3. New York

Unordered lists

An unordered, nested list looks like that:
  • Europe
    • London
    • Paris
  • America
    • New York
    • Houston

Description lists

A description list is used in a dictionary, for example. Each entry consists of a description title, and content.
Berlin
Charlottenburg
Treptow
Cape Town
Observatory
Rondebosch
